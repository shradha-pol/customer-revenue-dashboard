# 📊 Customer Revenue Dashboard (Excel + SQL)

This project presents an **interactive dashboard built using Excel and SQL** to analyze customer revenue, identify trends, and generate actionable business insights.

---

## 🎯 Objective

The goal of this project is to:

- Analyze customer revenue performance  
- Identify key business trends  
- Understand customer segments and regional contributions  
- Support data-driven decision-making

## 📈 Visualizations Used

- 📉 Monthly Revenue Trend (Line Chart)  
- 🍩 Revenue Contribution by Segment (Donut Chart)  
- 📦 Product-wise Revenue Distribution (Bar Chart)  
- 🌍 Regional Revenue Contribution (Bar Chart)  
- 🏆 Top 5 Customers by Revenue (Horizontal Bar Chart)  

---

## 🛠 Tools & Technologies

- **SQL (MySQL)** → Data extraction & transformation  
- **Microsoft Excel** → Dashboard development  
- Pivot Tables & Pivot Charts  
- GETPIVOTDATA  
- Data cleaning & formatting  

---

## 🔄 Workflow

1. Extracted raw data using SQL queries  
2. Performed data cleaning and transformation  
3. Imported dataset into Excel  
4. Created Pivot Tables and calculated metrics  
5. Built interactive dashboard with slicers and KPIs  

---

## 🎯 Key Insights

- Consumer segment contributes the highest revenue  
- South region generates maximum revenue  
- Electronics category leads in sales  
- A small group of customers drives a large portion of revenue  

---

## 🚀 Features

- Interactive slicers (Region, Segment, Product Category)  
- Dynamic KPI updates  
- Clean and professional UI design  
- Business-focused insights

## 📌 Conclusion

This dashboard helps stakeholders to:

- Monitor revenue performance  
- Identify high-performing segments and regions  
- Make informed business decisions
